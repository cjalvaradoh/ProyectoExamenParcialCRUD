
Proyecto CRUD - Evaluación Parcial

Objetivo
Este proyecto consiste en una aplicación de escritorio desarrollada en C# con Windows Forms, que permite realizar operaciones CRUD (Crear, Leer, Modificar, Eliminar) sobre la entidad `Producto`, aplicando los principios de la Arquitectura Limpia (Clean Architecture).

Estructura del proyecto
- Domain: contiene la clase Producto.
- Application: incluye los casos de uso para ventas.
- FormulariosVista: contiene los formularios gráficos como FrmProducto y MenuPrincipal.

Funciones del formulario de productos
- Crear un nuevo producto
- Mostrar productos en una tabla
- Modificar un producto seleccionado
- Eliminar un producto
- Limpiar los campos del formulario

Tecnología utilizada
- Lenguaje: C# (.NET)
- Interfaz: Windows Forms (WinForms)
- Patrón: Clean Architecture
- IDE: Visual Studio

Instrucciones de uso
1. Abrir el archivo .sln con Visual Studio.
2. Ejecutar el proyecto (F5).
3. Desde el menú principal, ingresar a la opción Productos.
4. Usar los botones para crear, modificar, eliminar o limpiar productos.

Estado del proyecto
El formulario de productos está completo y funcional, con operaciones CRUD conectadas a una lista de productos en memoria. Las vistas están organizadas y los eventos correctamente enlazados.
